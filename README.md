# Hotel Booking System

A web-based hotel booking system that allows users to browse rooms, make reservations, and manage their bookings.

## Features

- User dashboard for managing bookings
- Room browsing and selection
- Photo gallery of hotel rooms and facilities
- User profile management
- Interactive booking interface

## Pages

- `Dashboard.html` - Main user dashboard
- `Booking.html` - Room booking interface
- `Rooms.html` - Available rooms listing
- `Gallery.html` - Photo gallery
- `Profile.html` - User profile management

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

1. Clone this repository
2. Open `Dashboard.html` in your web browser to access the main interface
3. Navigate through different pages using the navigation menu

## Project Structure

```
├── Dashboard.html     # Main dashboard interface
├── Booking.html      # Booking management
├── Gallery.html      # Photo gallery
├── Profile.html      # User profile
├── Rooms.html        # Room listings
├── script.js         # JavaScript functionality
└── style.css         # Styling
```
